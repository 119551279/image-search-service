# Image Search Service

## Project Documentation

This project is an image search service that allows users to search for images using various filters and criteria.

## Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/119551279/image-search-service.git
   cd image-search-service
   ```

2. **Install dependencies**:
   Make sure you have Node.js installed, then run:
   ```bash
   npm install
   ```

3. **Run the application**:
   Start the application using:
   ```bash
   npm start
   ```

4. **Access the service**:
   Open your browser and navigate to `http://localhost:3000` to access the image search service.

## Features
- Search for images based on keywords
- Filter results by various parameters
- User-friendly interface

## Technologies Used
- Node.js
- Express
- MongoDB
